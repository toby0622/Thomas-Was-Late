# Thomas-Was-Late

## Keyboard Control

* Escape: Game Exit.
* WAD: Player 1 Control.
* Arrow Key: Player 2 Control.
* Q: Change Player (Single Player Mode).
* E: Split Screen (Multi Player Mode).
